## Avian skeletal ancestral-sequence reconstruction (ASR) quickstart

This compact workflow mirrors a successful run for avian skeletal-development genes (for example: `RUNX2`, `SHH`, `SOX9`, `FGF10`, `ALX1`, `TBX5`, `BMP4`, `HOXA11`, `FGF8`):

1. Download orthologs (`datasets`) and filter by target taxa (`seqkit`).
2. Build per-gene protein MSAs (`mafft`) and trim alignments.
3. Concatenate genes into a supermatrix and infer an ML tree (`iqtree`/`iqtree2`).
4. Run ASR and summarize branch-level amino-acid substitutions.
5. Prioritize branches with multi-gene signals and map internal nodes to descendant tips before interpretation.

### Practical notes

- If `datasets download gene gene-id ...` fails with *unable to parse input value as an integer: ''*, verify your gene table has **two columns** (`GENE` and numeric `ID`) separated by tab/space.
- In some environments the binary is `iqtree` instead of `iqtree2`; check with `which iqtree2` and `which iqtree`.
- Large substitution bursts should be treated as candidates pending alignment/orthology QC.
